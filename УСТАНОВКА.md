# Установка и запуск ReelsMaker Pro

## Требования
- Python 3.8+
- FFmpeg (должен быть в папке `ffmpeg/bin/`)

## Установка зависимостей

```bash
pip install -r requirements.txt
```

### Список зависимостей:
- PyQt5 - графический интерфейс
- qtawesome - иконки
- openai-whisper - транскрибация аудио
- g4f - бесплатный доступ к GPT-3.5
- google-api-python-client - YouTube API
- google-auth-oauthlib - OAuth для YouTube
- google-auth-httplib2 - HTTP клиент
- yt-dlp - скачивание видео с YouTube

## Запуск

```bash
python main.py
```

## Возможные проблемы

### 1. ModuleNotFoundError: No module named 'qtawesome'
**Решение:**
```bash
pip install qtawesome
```

### 2. FFmpeg не найден
**Решение:**
- Скачайте FFmpeg с официального сайта
- Распакуйте в папку `ffmpeg/bin/`
- Или добавьте FFmpeg в системный PATH

### 3. Ошибка импорта g4f
**Решение:**
```bash
pip install --upgrade g4f
```

### 4. Whisper требует больше памяти
**Решение:**
- Используйте модели "tiny" или "base" вместо больших
- Закройте другие приложения

## Новые функции

### ИИ Авто-нарезка
1. Перейдите в "ИИ Авто-нарезка"
2. Загрузите длинное видео
3. Укажите диапазон анализа
4. Настройте параметры
5. Запустите обработку

### Предпросмотр субтитров
1. Добавьте видео в список
2. Выберите "Из файла SRT"
3. Укажите .srt файл
4. Нажмите "Предпросмотр стиля"
5. Настройте и примените

```
├── ui/

│   ├── subtitle_preview_dialog.py  # Предпросмотр субтитров

├── utils/
│   ├── ai_clipper.py      # ИИ-нарезка видео
│   ├── audio_uniquify.py  # Аудио-уникализация
│   ├── advanced_filters.py # Продвинутые фильтры
│   ├── subtitle_utils.py  # Утилиты субтитров
│   ├── ffmpeg_utils.py    # FFmpeg обработка
│   └── constants.py       # Константы
├── workers/
│   └── worker.py          # Воркер обработки
├── resources/
│   ├── styles_dark.qss    # Темная тема
│   └── styles_light.qss   # Светлая тема
└── ffmpeg/
    └── bin/
        ├── ffmpeg.exe
        └── ffprobe.exe
```

## Поддержка
При возникновении проблем проверьте:
1. Установлены ли все зависимости
2. Правильно ли указан путь к FFmpeg
3. Достаточно ли оперативной памяти
4. Наличие интернета для g4f (ИИ-нарезка)
