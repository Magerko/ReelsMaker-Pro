# ReelsMaker Pro

Мощный инструмент для массовой обработки видео под формат Reels/TikTok/Shorts с поддержкой фильтров, субтитров и аудио-наложения.

## Возможности

- **Массовая обработка** — обрабатывайте несколько видео за раз
- **Формат Reels/TikTok** — автоматическое преобразование в вертикальный формат 1080x1920
- **Размытый фон** — заполнение черных полос размытой версией видео
- **30+ фильтров** — цветокоррекция, VHS-эффект, виньетка, шум и многое другое
- **Субтитры** — генерация через Whisper AI или загрузка из SRT
- **Наложение изображений** — добавление водяных знаков, логотипов (PNG, JPG, GIF)
- **Наложение аудио** — добавление фоновой музыки с регулировкой громкости
- **Трансформации** — зум, скорость, отражение, обрезка черных полос
- **Аппаратное ускорение** — поддержка NVIDIA, Intel, AMD кодеков
- **Темная и светлая тема**

## Установка

### Требования
- Python 3.8+
- FFmpeg

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Magerko/ReelsMaker-Pro.git
cd ReelsMaker-Pro
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Скачайте [FFmpeg](https://ffmpeg.org/download.html) и поместите в папку `ffmpeg/bin/` или добавьте в системный PATH.

4. Запустите приложение:
```bash
python main.py
```

## Использование

1. **Добавьте видео** — перетащите файлы или используйте кнопку "Добавить видео"
2. **Настройте параметры** — выберите формат, фильтры, субтитры
3. **Обработайте** — нажмите "Обработать" и выберите папку для сохранения

### Вкладки настроек

| Вкладка | Описание |
|---------|----------|
| Меню | Формат вывода, кодек, предпросмотр |
| Трансформация | Обрезка, фильтры, зум, скорость |
| Наложение | Изображения, субтитры |
| Аудио | Громкость, фоновая музыка |

## Поддерживаемые форматы

**Входные:** MP4, MOV, AVI, MKV, FLV, WMV, GIF

**Выходные:** Оригинальный формат или Reels/TikTok (1080x1920)

## Зависимости

- PyQt5 — графический интерфейс
- qtawesome — иконки
- openai-whisper — генерация субтитров
- FFmpeg — обработка видео
